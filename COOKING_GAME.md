# 🍳 AI Cooking Simulation - Fish Cake Kitchen

## 🎯 ゲーム概要

アイソメトリック視点の料理シミュレーションゲームです。3人のAIシェフが協力してフィッシュケーキを作ります。

## 🎮 ゲーム機能

### **👨‍🍳 AI料理人**
- **3人のシェフ**: 白いコック服とコック帽
- **協力プレイ**: 各シェフが異なるタスクを分担
- **自動行動**: 材料取得→調理→配膳まで自動化

### **🏪 キッチン設備**
- **冷蔵庫**: 材料保管（魚の腹身など）
- **調理台**: 材料をカットする作業場
- **コンロ**: フライパンで炒める
- **オーブン**: 焼き上げ作業
- **シンク**: 洗い物
- **配膳エリア**: 完成品を置く場所

### **📋 レシピシステム**
- **フィッシュケーキ**作りのワークフロー:
  1. **材料取得**: 冷蔵庫から魚の腹身を取る
  2. **切る**: 調理台で材料をカット
  3. **炒める**: コンロで調理
  4. **焼く**: オーブンで仕上げ

### **🎨 UIシステム**
- **タスクリスト**: 左上に表示される進行状況
- **進行表示**: 
  - ✓ 完了タスク（緑色）
  - ◇ 現在のタスク（青色） 
  - ◇ 未来のタスク（灰色）

## 📐 技術仕様

### **視覚スタイル**
- **アイソメトリック視点**: 斜め上から見下ろし
- **ボクセル風グラフィック**: 温かみのある3Dブロック
- **室内照明**: 暖色系の優しいライティング

### **カメラシステム**
- **回転カメラ**: キッチン全体を自動で見回す
- **最適な視点**: 全設備が見えるアングル

### **パフォーマンス**
- **軽量設計**: シンプルなジオメトリ
- **効率的な材質**: 反射計算を最小化

## 🔄 ゲームフロー

```
開始 → シェフ配置 → 材料取得 → カット → 炒め → 焼き → 完成 → リピート
```

### **AIの行動パターン**
1. **タスク取得**: レシピから現在のステップを確認
2. **移動**: 必要な設備へ自動移動
3. **作業実行**: 設備に到着したら作業開始
4. **完了**: 次のタスクまたは次のシェフに引き継ぎ
