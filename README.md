# Agentira

Multi-agent factory simulation game for educational purposes.

## Overview

Agentira is an educational serious game that teaches AI agent interaction and factory automation concepts. Inspired by games like Factorio, it provides a visual environment where students can learn about:

- Multi-agent systems
- Resource management
- Process optimization
- Agent cooperation and competition
- Workflow automation

## Features

- **Multi-agent simulation**: Multiple AI agents working together
- **Educational focus**: Learn by doing approach
- **Visual interface**: Real-time visualization of agent behavior
- **Factory simulation**: Resource gathering, production, and optimization
